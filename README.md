# Название проекта: SnowfallGameFNA
- - - -
#### 2D-симуляция снегопада в деревне на фреймворке FNA.
## Описание
Проект демонстрирует реалистичный снегопад на фоне зимнего пейзажа. Скорость каждой снежинки зависит от её размера, создавая эффект глубины.
## Структура проекта
Constants: размеры, количество снежинок, скорость; Content: текстуры;
Модель данных для снежинки (SnowFlakeModels); Нативные библиотеки FNA (SDL3, FNA3D); Snowfall.cs Основная логика игры (инициализация, обновление, отрисовка).
